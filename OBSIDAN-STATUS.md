## 💡 实践原则

### 新笔记创建时
```markdown
# 笂记标题

相关笔记：
- [[相关笔记1]]
- [[相关笔记2]]

内容...
```

### 更新笔记时
```markdown
想到相关内容 → 立即添加 [[链接]]
发现新关联 → 添加双向链接
```

### 回顾笔记时
```markdown
跟随链接探索 → 发现新关联
补充缺失链接 → 强化网络
```

## 🎓 学习记录

**关键洞察**:
- 链接不仅是指向，更是思考
- 知识网络比孤立笔记更有价值
- 意外的关联带来新的洞察

**遇到的问题**:
- 初期容易忘记添加链接
- 链接什么内容需要判断
- 过度链接 vs 不足链接的平衡

**解决方案**:
- 新笔记强制添加至少1个链接
- 更新笔记时主动寻找关联
- 定期回顾和优化
