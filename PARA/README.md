# PARA知识管理系统 - 贾维斯

## 系统概述

**PARA Method** by Tiago Forte:
- **P**rojects (项目) - 当前正在进行的工作，有明确截止日期
- **A**reas (领域) - 长期责任范围，没有固定结束时间
- **R**esources (资源) - 主题或兴趣，持续积累
- **A**rchives (归档) - 非活跃项目，保留参考

**Zettelkasten (卡片记忆法)**:
- 每张卡片一个想法
- 建立卡片之间的链接
- 累积知识复利

---

## 目录结构

```
PARA/
├── Projects/          # 活跃项目
├── Areas/            # 责任领域
├── Resources/        # 知识资源
└── Archives/         # 已归档

Zettelkasten/         # 永久笔记
```

---

## 使用原则

1. **项目完成后归档到Archives**
2. **领域持续维护，不归档**
3. **资源按主题分类**
4. **永久笔记记录洞察和模式**
5. **建立跨文件链接**（使用`[[链接]]`格式）
