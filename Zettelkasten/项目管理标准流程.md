## ⚡ 自动化脚本

我需要创建一个脚本来自动化这个流程：

```python
# create_project.py
import os
import json
from datetime import datetime

def create_project(name):
    """创建标准项目结构"""

    base_path = f"/home/ubuntu/.openclaw/workspace/PARA/Projects/{name}"

    # 创建目录结构
    dirs = [
        "截止日期跟结果描述",
        "项目任务",
        "项目闪念",
        "这个项目的文件/脚本",
        "这个项目的文件/文档",
        "这个项目的文件/日志"
    ]

    for d in dirs:
        os.makedirs(f"{base_path}/{d}", exist_ok=True)

    # 创建README.md
    readme_content = f"""# {name}

**创建时间**: {datetime.now().strftime('%Y-%m-%d')}
**状态**: 🔄 进行中
**优先级**: ⭐⭐⭐

---

## 项目概述

简要描述项目的目标、背景。

---

## 1. 截止日期跟结果描述

### 时间线
- **开始日期**: {datetime.now().strftime('%Y-%m-%d')}
- **截止日期**: 待定
- **实际完成**: 待定

### 交付成果
- [ ] 成果1: 待定
- [ ] 成果2: 待定
- [ ] 成果3: 待定

---

## 2. 项目任务

- [ ] [[task-01-待定]]
- [ ] [[task-02-待定]]
- [ ] [[task-03-待定]]

---

## 3. 项目闪念

- [[闪念-待定]]
- [[闪念-待定]]

---

## 4. 这个项目的文件

- [[脚本/待定]]
- [[文档/待定]]
- [[日志/待定]]

---

**最后更新**: {datetime.now().strftime('%Y-%m-%d %H:%M')}
"""

    with open(f"{base_path}/README.md", 'w') as f:
        f.write(readme_content)

    print(f"✅ 项目 '{name}' 已创建在 {base_path}")
```
